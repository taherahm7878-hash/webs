First Lesson

How to print text and numbers?

To print number you need to type

console.log( ); or / او document.write( );

لطباعة رقم انت تحتاج لكتابة

Simple example

console.log(200);
document.write(200);

To print text you need to type

console.log(" "); or / او document.write(" ");

لطباعة نص انت تحتاج لكتابة

Simple example

console.log("Banana");
document.write("Banana");

Remember that !!

!! تذكر أن 

 
 To print a text, write :
console.log(" Text ");
OR
document.write(" Text ");

 : لطباعة نص تكتب 
console.log(" Text ");
document.write(" Text ");

  To print a number, write :
console.log( Number );
document.write( Number );

 : لطباعة رقم تكتب 
console.log( Number );
document.write( Number );
