How to declare a variable?

To declare a variable, you need to write

let fish = (number or string);

للاعلان عن متغير انت تحتاج لكتابة

Simple example

let fish = 10;
let fish = "10";

To declare a constant variable you need to write

 


const day = 7;
const meter = 100;

Notice

لاحظ

that the number of days in a week is constant and does not change.

ان عدد الايام في الاسبوع ثابتة لا تتغير

Remember that !!

!! تذكر أن 

 
To declare a variable that changes, we write :
let price = 1300;

للاعلان عن متغير يتغير نكتب :
let numberofrabbits = 1790;

To declare a variable that does not change, we write :
const numberofeyes = 2;

للاعلان عن متغير لا يتغير نكتب :
const numberofeyes = 2;
